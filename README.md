Dream Adventures – Kenya Tourism App
A modern, responsive web application designed to help travelers discover, explore, and book unforgettable safari and coastal experiences across Kenya. Built with pure frontend technologies and integrated with live travel data via RapidAPI.
Key Features
🔍 Live Destination Search – Fetch real-time Kenyan travel locations using the Travel Advisor API
🛡️ Smart Fallback System – Automatically switches to curated local destinations if the API fails, times out, or hits rate limits
📅 Interactive Booking Flow – Clean form with validation + success confirmation modal
🖼️ Safari Gallery & Blogs – Curated photo showcase and traveler ratings section
📱 Fully Responsive – Optimized for mobile, tablet, and desktop screens
⚡ Zero Dependencies – Built with vanilla HTML, CSS, and JavaScript for maximum performance
